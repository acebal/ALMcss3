Collapsing Margins

This is an example of how collapsing margins work in CSS. And it is what explains the apparently strange behaviour on the former columns-000.html example (which now has also been added, slightly modified, as the test case collapsing-margins-001.html . Here, using just CSS 2.1 properties, we are emulating what would be the generated grid in that case, being #grid1 the automatically created div for the template (or grid), and #slot1 that of the slot containing the h1 element.
